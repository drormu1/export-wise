# Task Execution Checklist

## Before Start
- [ ] Task file exists: `tasks/<task-name>.md`
- [ ] Scope is clear (in/out)
- [ ] Required context/docs identified
- [ ] Constraints are explicit

## During Implementation
- [ ] Follow project rules from `CLAUDE.md` and `context.md`
- [ ] Keep changes minimal and task-focused
- [ ] Avoid unrelated refactors
- [ ] Validate incrementally (not only at the end)

## Before Finish
- [ ] Acceptance criteria reviewed and satisfied
- [ ] Verification commands executed
- [ ] Results documented
- [ ] Risks/assumptions documented
- [ ] Changed files list prepared

## Optional Git Readiness
- [ ] Commit message drafted
- [ ] PR summary prepared
